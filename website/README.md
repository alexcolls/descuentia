# Descuentia Corporate Website

This directory contains the marketing website for Descuentia, built with **Next.js 14**, **TypeScript**, **Tailwind CSS**, and **Framer Motion**.

## Getting Started

```bash
cd website
npm install
npm run dev
```

Open `http://localhost:3000` to view the site in development mode.

## Scripts

- `npm run dev` – Start the development server
- `npm run build` – Create an optimized production build
- `npm run start` – Run the production server
- `npm run lint` – Lint the project with ESLint

## Deployment

1. Build the site: `npm run build`
2. Start the production server: `npm run start`

You can deploy to any platform that supports Node.js (Vercel, Netlify, AWS, etc.).

## Tech Stack

- **Framework:** Next.js 14 (App Router)
- **Languages:** TypeScript, JSX
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion

## Environment

This marketing site is static-first and does not require environment variables. Update metadata in `app/layout.tsx` and content assets in `lib/data.ts`.
